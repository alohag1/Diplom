# Терем ок?

Мультиагентная система анализа рекламных креативов на базе локальных AI-моделей.

Загрузите изображение — система оценит его по 4 критериям (типографика, цвет, иерархия, композиция), выдаст оценки и рекомендации по улучшению.

## Архитектура

| Компонент | Модель | Назначение |
|---|---|---|
| LLM | qwen3.5:9b (Ollama) | Анализ, оценки, рекомендации |
| Визуальный поиск | CLIP (clip-vit-base-patch32) | Поиск похожих из датасета, объективная оценка |
| RAG | nomic-embed-text + ChromaDB | Поиск знаний из книг по дизайну |
| Метрики | image_analyzer.py (Python) | Извлечение цветов, контраста, композиции |

### Логика работы

1. **CLIP** находит 10 визуально похожих изображений из датасета (2363 эталонных примера good/bad)
2. **image_analyzer** извлекает числовые метрики (палитра, контраст, баланс)
3. **RAG** находит релевантные фрагменты из книг по дизайну (1740 фрагментов из 6 книг)
4. **qwen3.5:9b** получает всё вместе и генерирует оценки + рекомендации

Время анализа: **~30 секунд**.

## Стек

- Python 3.14, FastAPI, Ollama
- ChromaDB, LangChain, PyTorch, CLIP (HuggingFace Transformers)
- Датасет: 2363 размеченных рекламных креатива (good/bad)
- База знаний: 6 книг по дизайну, типографике, цветоведению

## Установка и запуск

### Требования

- Python 3.11+
- [Ollama](https://ollama.com/download)

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/alohag1/Diplom.git
cd Diplom

# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
.\venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Скачать модели Ollama
ollama pull qwen3.5:9b
ollama pull nomic-embed-text
```

### Подготовка данных

```
data/
  books/           # PDF-книги по дизайну
  dataset/
    good/          # Хорошие примеры
      posters/
      web_design/
      typography/
    bad/           # Плохие примеры
      posters/
      web_design/
      typography/
```

```bash
# Загрузить книги в RAG-базу
python ingest.py

# Создать CLIP-индекс датасета
python build_clip_index.py

# Загрузить метрики датасета
python ingest_dataset.py
```

### Запуск

```bash
python api.py
```

Открыть: **http://localhost:8000**

## API

| Метод | Путь | Описание |
|---|---|---|
| GET | `/api/health` | Статус сервера |
| POST | `/api/analyze` | Загрузить изображение, получить анализ |
| POST | `/api/analyze/base64` | Анализ изображения в base64 |
| POST | `/api/ingest-dataset` | Запустить загрузку датасета |

## Критерии оценки

- **Типографика** — шрифты, читаемость, контраст текста
- **Цветовые решения** — палитра, гармония, контрастность
- **Визуальная иерархия** — фокус внимания, логика восприятия
- **Композиция** — баланс, пропорции, использование пространства
